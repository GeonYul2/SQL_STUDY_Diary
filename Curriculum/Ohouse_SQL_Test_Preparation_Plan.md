# 오늘의집 SQL Test 준비 플랜

## 목표

프로그래머스 플랫폼 적응 + 오늘의집 포지션 특화 SQL 역량 + 영어/한국어 문제 적응을 동시에 준비한다.

## 준비 축 4개

### 1. 플랫폼 적응: `Problems/Programmers/`

목적:
- 프로그래머스 출력 형식에 익숙해지기
- 제한 시간 내 풀이 감각 확보
- 자주 나오는 유형 반복

우선 유형:
- SELECT
- JOIN
- GROUP BY
- IS NULL
- SUM, MAX, MIN

### 2. 포지션 특화: `Problems/Self-Made/6_채용대비/오늘의집_SQL_Test/`

목적:
- 로그 품질 검증
- 행동 로그 분석
- funnel / retention / latest-state / anomaly detection

우선 유형:
- null / duplicate / integrity check
- 단계별 전환율
- 일자별 이벤트 급감/급증
- 최신 이벤트 기준 상태 추출
- window function 기반 사용자 분석

### 3. 언어 적응: 영어 + 한국어

목적:
- 영어 문제 서술 해석 속도 향상
- 한국어 설명과 영어 설명을 매칭해 SQL 사고를 고정

실행 방식:
- Self-Made mock 문제는 반드시 한국어/영어 동시 작성
- 풀이 해설도 가능하면 두 언어로 짧게 정리
- 치트시트에는 핵심 영어 표현을 함께 기록

### 4. 취약점 보완: `Curriculum/Ohouse_SQL_Test_Review_Queue.md`

목적:
- 이미 막혔던 문제를 다시 풀어 정확도 올리기
- 시간 압박 상황에서 같은 실수 반복 방지
- 시험 직전 안정성 확보

실행 방식:
- `[재풀이 필요]` 표기 문제를 우선 관리
- 프로그래머스 / Self-Made를 함께 보되, 시험 직전에는 프로그래머스를 우선
- 단순 정답 복기가 아니라 **가상 풀이 -> 검토 -> 재작성** 루프로 진행

## 권장 진행 순서

1. 프로그래머스 2문제
2. 오늘의집 특화 mock 1문제

4. 치트시트 보강
5. 영어 표현 복습
